    # Neon UI design

    This library provides production-oriented React components, CSS custom properties, and TypeScript tokens for frontend teams building product interfaces.

    ## Principles

    - Keep controls predictable, compact, and accessible.
    - Prefer native semantics where possible.
    - Use stable dimensions for buttons, inputs, tabs, and overlays.
    - Treat Storybook as the shared source of truth for states and variants.

    ## Import

    ```tsx
    import { Button, TextField, Select } from "neon-ui-lib";
    ```

    ## CSS

    Import once at the app root if your bundler does not automatically include package CSS:

    ```tsx
    import "neon-ui-lib/styles.css";
    ```
